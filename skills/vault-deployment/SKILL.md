description: Deploy and configure CreatorVault smart contract vaults across supported networks.

## Activation
- Confirm target network, deployer wallet, and required environment variables are available.
- Validate deployment inputs (vault metadata, fee settings, role addresses) with stakeholders.

## Assets
- Deployment scripts in `script/` and contract artifacts from `contracts/`.
- Network RPC endpoints, block explorer links, and multisig addresses.
