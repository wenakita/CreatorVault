## FAQ Section

### Q1: What is Eagle?
**A**: Eagle is an omnichain fungible token (OFT) that can be bridged across Ethereum, Base, BNB Chain, and Solana using LayerZero V2. It represents liquid vault shares that earn yield.

### Q2: How long do cross-chain transfers take?
**A**: Most transfers complete in 10-20 minutes. The exact time depends on block confirmations and LayerZero DVN verification.

### Q3: What is the 1-Transaction Compose feature?
**A**: It allows you to send EAGLE from Base and automatically receive WLFI back on Base‚Äîall in one transaction. The system handles unwrapping, redemption, and bridging automatically.

### Q4: Is Eagle secure?
**A**: Yes! Every cross-chain message is verified by 2 independent DVNs (Google Cloud and LayerZero Labs). All contracts are open-source and verified on block explorers.

### Q5: What happens if a transaction fails?
**A**: ComposerV2 has built-in automatic refunds. If any operation fails, your EAGLE tokens are automatically refunded to your address.

### Q6: What is the 10,000x dilution factor?
**A**: The EagleOVault was intentionally diluted by 10,000x to enable the full EAGLE token supply. This means 1 EAGLE ‚âà 0.0001 WLFI when redeemed.

### Q7: Can I trade Eagle tokens?
**A**: Yes! Eagle is a standard ERC20 token on EVM chains and can be traded on any DEX or CEX that lists it.

### Q8: Which wallets are supported?
**A**: Any wallet that supports the respective chain. For EVM chains: MetaMask, WalletConnect, Coinbase Wallet, etc. For Solana: Phantom, Solflare, etc.
